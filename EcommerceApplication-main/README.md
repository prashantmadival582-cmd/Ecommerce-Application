# 🛒 E-Commerce Application

## 📌 Project Overview

The E-Commerce Application is a web-based application developed using Java and Spring Boot. It provides functionalities for managing products, users, and orders, enabling a seamless online shopping experience.

## 🚀 Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* REST APIs
* Hibernate

## ✨ Features

* User Registration and Login
* Product Management
* Product Search
* Shopping Cart Management
* Order Placement
* Database Integration with MySQL
* RESTful API Architecture

## ⚙️ Prerequisites

* Java 17 (or your version)
* Maven
* MySQL Server
* IDE (Eclipse / IntelliJ IDEA / VS Code)

## 🔧 Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/prashantmadival582-cmd/Ecommerce-Application.git
```

2. Configure MySQL database in `application.properties`.

3. Build the project:

```bash
mvn clean install
```

4. Run the application:

```bash
mvn spring-boot:run
```

## 📡 API Endpoints

| Method | Endpoint       | Description       |
| ------ | -------------- | ----------------- |
| GET    | /products      | Get all products  |
| GET    | /products/{id} | Get product by ID |
| POST   | /products      | Add a product     |
| PUT    | /products/{id} | Update product    |
| DELETE | /products/{id} | Delete product    |
